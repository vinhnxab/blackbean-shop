Products, cart, checkout, orders, PayPal payments, and license sales for Blackbean sites.

== Description ==

Black Bean Shop provides a lightweight e-commerce layer without WooCommerce:

* Product catalog at `/products/`
* Session cart and checkout at `/shop/cart/` and `/shop/checkout/`
* Order management in wp-admin
* PayPal integration
* Digital downloads and license activation API

Requires the **Black Bean Tables** plugin.

== Installation ==

1. Upload and activate **Black Bean Tables**, then **Black Bean Shop**.
2. Visit **Settings → Permalinks** and save once (or deactivate/reactivate the plugin).
